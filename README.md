# 🔧 Workshop Management App (Flutter + Firebase)

## 🚀 Overview

This mobile application helps workshop owners manage customer and vehicle records digitally. It replaces traditional paper-based systems with a smart and efficient solution.

## ✨ Features

* Customer Information Management
* Vehicle Details Storage
* Job Card Creation & Tracking
* Real-time Data Updates
* Easy Record Management

## 🛠️ Tech Stack

* Flutter
* Firebase (Firestore, Authentication)

## 🎯 Problem Solved

Many workshop owners struggle with managing records manually. This app digitizes all data, making it easy to store, update, and track work efficiently.
